# Тест-кейсы: Управление клиентами в Banking App

## TC-01: Создание клиента (Add Customer)

**Предусловие:**  
Открыт сайт https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager

**Шаги:**
1. Нажать кнопку "Bank Manager Login"
2. Нажать кнопку "Add Customer"
3. Сгенерировать Post Code (10 цифр)
4. Сгенерировать First Name на основе Post Code:
    - Разбить Post Code на 5 двузначных чисел
    - Каждое число преобразовать в букву (0=a, 26=a, 52=a и т.д.)
5. Ввести First Name, Last Name (любое), Post Code
6. Нажать "Add Customer"
7. Проверить, что появилось уведомление "Customer added successfully"

**Ожидаемый результат:**  
Клиент создан, появилось уведомление об успешном добавлении.

## TC-02: Фильтрация клиентов по буквам (поиск)

**Предусловие:**  
Добавлено не менее 2 клиентов, включая тестовых

**Шаги:**
1. Перейти в раздел "Customers"
2. Получить исходный список имён клиентов
3. Найти поле поиска (фильтр) по имени
4. Поочерёдно ввести буквы имени одного из клиентов (например, "Harry")
5. После каждого ввода убедиться, что список обновляется динамически
6. Проверить, что в списке остаются только клиенты, чьи имена начинаются с введённой последовательности

**Ожидаемый результат:**  
Список клиентов фильтруется в реальном времени. После ввода полного имени — остаётся только один клиент.

## TC-03: Удаление клиента с именем, ближайшим к средней длине

**Предусловие:**  
Добавлено не менее 3 клиентов

**Шаги:**
1. Перейти в раздел "Customers"
2. Получить список имён клиентов
3. Вычислить среднюю длину имён с помощью `Stream API`:
4. Найти имя, длина которого наименее отличается от средней
5. Найти строку с этим именем и нажать кнопку "Delete"
6. Проверить, что клиент отсутствует в таблице

**Ожидаемый результат:**  
Клиент с именем, ближайшим к средней длине, успешно удалён из списка.
